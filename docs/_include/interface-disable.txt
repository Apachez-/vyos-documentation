  {{ var5 }} {{ var6 }} disable

  **Disable the interface.**

  The interface will be set to the administratively down
  (``A/D``) state.

  Example:

  .. code-block:: none

    set interfaces {{ var0 }} {{ var1 }} {{ var2 }} {{ var4 }} {{ var5 }} {{ var7 }} disable
